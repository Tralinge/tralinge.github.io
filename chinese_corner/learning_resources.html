Mango Languages
Great for Listening, Reading
Mango Languages is currently my favorite general language app. With real speakers, interactive script (Chinese characters with pinyin on hover), and lessons presented as conversation that is built upon previous vocabulary, this app is a great resource for learning. There are roughly 500 lessons, as well as some supplementary Mandarin courses discussing fengshui and the Chinese zodiac. I assume this program is typically expensive, but you can access it through almost any local library for free (make an account for the online resources at your library and sign in).
